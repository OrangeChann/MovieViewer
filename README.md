# Movie Viewer

Android movie browsing app built with Kotlin and Jetpack Compose, 
powered by the TheMovieDB API.

## Features
- User authentication with persistent local storage
- Browse movies by Popular, Top Rated, Now Playing and Upcoming
- Movie detail view with full metadata
- User reviews per movie
- Favourite movies list stored locally per user
- Movie search by title
- Similar movie recommendations
- Offline support via Room database

## Tech Stack
- Kotlin
- Jetpack Compose
- Retrofit (REST API)
- Room (local database)
- ViewModel + Coroutines
- TheMovieDB API
- Android API 34

## UI/UX Highlights
- Personalised greeting on nav bar
- Search bar for quick movie discovery
- Expandable reviews with "Read More" for clean layout
- Card-based favourite movies list
- Form validation with inline error messages
- Consistent design language across all screens

## Setup
1. Clone the repo
2. Open in Android Studio
3. Add your TheMovieDB API key
4. Run on Pixel 8 Pro emulator (API 34)
